# Contributing

Contributions are welcome in the [library repository](https://github.com/rutviknabhoya/react-native-alternate-app-icon):

- [Contributing guide](https://github.com/rutviknabhoya/react-native-alternate-app-icon/blob/main/CONTRIBUTING.md)
- [Code of Conduct](https://github.com/rutviknabhoya/react-native-alternate-app-icon/blob/main/CODE_OF_CONDUCT.md)
- [Open an issue](https://github.com/rutviknabhoya/react-native-alternate-app-icon/issues/new/choose)

For major changes, open an issue first so we can align on the approach.
