# Installation

Install the package together with its **required** peer dependency. Pin **`react-native-nitro-modules` to 0.32.0** — see [Compatibility](/docs/getting-started/compatibility) for the full version matrix.

<InstallPackages />

### iOS — CocoaPods

After installing JavaScript dependencies, install native pods:

```bash
cd ios && pod install
```

### Rebuild native apps

Icon changes require native configuration. After setup, rebuild your app:
